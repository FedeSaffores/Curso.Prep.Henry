Introduccin a HTML

HTML es el bloque bsico con el que est construido internet. Todas las pginas web utilizan HTML. No es un lenguaje de programacin propiamente dicho, sino, es un lenguaje de Markup: son lenguajes que incorporan al texto marcas o etiquetas que luego son interpretadas para darle informacin extra sobre la estructura del texto. En el caso de HTML, este ser interpretado por los browsers, que tambien presentaran el cdigo en forma grfica.

HTML es la abreviatura de Hyper Text Markup Language:

  • Hyper Text: Hyper Texto quiere decir texto con links
  • Markup Language: Los Lenguajes de Marcado son lenguajes de programacin basados en etiquetas que uno agrega a un texto para darle estructura e informacin adicional. A diferencia de los Lenguajes de Scripting que se usan para crear programas informticos, los lenguajes de marcado son slo reglas para ordenar un documento.

Elementos bsicos HTML

HTML define una serie de elementos (o etiquetas, o tags) que serviran para delimitar texto. Cada tag est encerrado en < > y tiene un nombre. Los tags se abren y se cierran, los tags de cerrado incluyen con un / en el principio del tag que cierra. Por ejemplo:

<element>
  ...
</element>

Algunos tags html, por su naturaleza, no necesitan tener nada adentro. Por lo tanto podemos abreviar su escritura y en vez de abrir y cerrar el tag, simplemente agregamos un / antes del bracket final.

<img src="http://imagen.com/img.jpg" />

Atributos

En su mayora de los atributos de un elemento son pares nombre-valor, separados por un signo de igual = y escritos en la etiqueta de comienzo de un elemento, despus del nombre del elemento. El valor puede estar rodeado por comillas dobles o simples. Los atributos de los tags nos sirven para cambiar su comportamiento o configurarlos.

Por ejemplo, el tag <img> sirve para mostrar una imagen. Este tag recibe el atributo src que indica la URL de donde est la imagen que queremos mostrar.

<img src="http://imagen.com/img.jpg" />

Introduccin a CSS

Como vimos, HTML nos sirve para dar estructura al contenido. En las primeras pocas de internet las pginas eran as. De hecho, todava esta online la primera pgina web. Como ven es bastante aburrida.
Luego se introdujo el concepto de CSS (Cascading Style Sheets); una forma de poder agregar color y estilos en nuestras pginas!

Reglas CSS

Bsicamente una regla CSS est compuesta por un atributo o propiedad y un valor. Seun el atributo que usemos y el valor que le pongamos a ese atributo vamos a obtener resultados visuales distintos en nuestro html.

Por ejemplo:

html {
  color: red; /* "Color" es la propiedad y "red" el valor */
  font-size: 12px; /* "font-size" es la propiedad y "12px" el valor */
}

En este ejemplo, vemos dos atributos: color y font-size, el primero permite modificar el color del texto, en este caso est seteado a red; el segundo indica el tamao del texto, en este caso 12px.
Es importante notar que distintos atributos pueden recibir distintos valores, generalmente los que indican un color reciben un color (red, blue, etc), los que son medidas reciben una medida (12px, 15px, etc), y hay otras propiedades que reciben valor especificos, por ejemplo: la propiedad border (que dibuja un borde alrededor de un elemento) recibe tres valores: el color del borde, el ancho de la linea y el tipo de linea (punteada, continua, etc).

Hay muchos atributos CSS disponibles, ms de los que podemos recordar. As que no se asusten, con el tiempo van a empezar a memorizar estas propiedades. Pueden ver una lista completa ac.

Formas de dar estilo

Antes de empezar a dar estilos, necesitamos una forma de decirle al browser qu vamos a darles reglas de estilo.
Hay varias formas de lograr esto (ms adelante veremos en detalle como funcionan cada una):

  • usando el atributo style: esta es la forma primitiva ms simple, bsicamente le damos reglas a cada tag html.
  • usando el tag <style/>: Se utiliza este tag en el <head> del documento HTML, con esto logramos agrupar todas las reglas que luego queremos que se apliquen a los elementos HTML.
  • Usar el tag <link/>: Este mtodo nos permite definir las reglas CSS en un documento separado e importarlo a nuestra pgina (la ventaja que tiene es que podemos importar el mismo CSS a varias pginas).

Atributo style

Todos los tags HTML pueden recibir el atributo style. Este atributo indica las reglas CSS (que veremos ms abajo), que se aplicaran slo al elemento que las tiene.

<h1 style="color:blue;">Esto es un ttulo Azul</h1>

Pros:

  • Fcil de escribir y leer.
  • Cmo se aplican a un slo elemento no hay forma de confundirse y que se aplique la regla a un elemento no deseado.

Cons:

  • La regla aplica a un slo elemento, si quisieramos que varios elementos tengan la misma regla, deberamos copypastear!

<style>

El tag <style>, que se escribe en el <head> del documento, nos permite escribir reglas que se aplicaran a uno o varios elementos html. Es importante notar que con esta forma, podremos darle estilo a muchos elementos de una sla vez, pero slo a elementos que estn en el mismo documento.

<html>
  <head>
    <style>
      /*<!-- ac van las reglas -->*/
    </style>
  </head>
  <body>
  </body>
</html>

Pros:

  • Lugar central donde podemos escribir las reglas CSS del documento
  • Podemos compartir reglas entre varios elementos iguales

Cons:

  • No podemos compartir las reglas con otro documento HTML.
  • Hay que prestar atencin a las reglas, y a qu elementos se aplican.

<link>

Con el tag <link> dentro del <head> del documento, vamos a poder importar un archivo css que contenga varias reglas CSS. Funciona similar al tag <style/> anterior. Pero ahora tenemos la ventaja que podemos compartir el mismo archivo css con varios documentos HTML.

<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
  </body>
</html>

Pros:

  • Lugar central donde podemos escribir las reglas CSS del documento
  • Podemos compartir reglas entre varios elementos iguales
  • Podemos compartir reglas entre varios documentos HTML

Cons:

  • Hay que prestar atencin a las reglas, y a qu elementos se aplican.

Selectores

Para poder aplicar reglas de estilo a los elementos html, necesitamos una forma de saber cmo seleccionar los elementos a los que deseamos aplicar las reglas, para esto sirven los selectores CSS.

Hay varios tipos de selectores, los ms bsicos son los de tipo, donde indicamos a qu clase de elemntos se van a aplicar las reglas, el ejemplo de arriba usa un selector de tipo. Est diciendo: aplicarle a todos los elementos de tipo <p> la regla de texto color rojo.

El selector de tipo se puede usar con cualquier tipo de tag: p, div, body, etc. Otra forma de usar selectors poniendole un nombre o identificador a cada elemento HTML. Para esto existe un atributo que pueden recibir todos los tags llamados: id y class.

<div id="divId"></div>
<div class="divClass"></div>

Id: son nombre que slo pueden aparecer una slo vez en el documento. Es super especifico y sirve para seleccionar UN solo elemento en particular.

Class: podemos asignarle el nombre de una clase a un grupo de elementos html.

Selectores bsicos

  • Selector de tipo: Selecciona todos los elementos que coinciden con el nombre del elemento especificado.
    Sintaxis: eltname
    Ejemplo: input se aplicar a cualquier elemento <input>.
  • Selector de clase:
    Selecciona todos los elementos que tienen el atributo de class especificado.
    Sintaxis: .classname
    Ejemplo: .index seleccionar cualquier elemento que tenga la clase index.
  • Selector de ID
    Selecciona un elemento basndose en el valor de su atributo id. Solo puede haber un elemento con un determinado ID dentro de un documento.
    Sintaxis: #idname
    Ejemplo: #toc se aplicar a cualquier elemento que tenga el ID toc.
  • Selector universal
    Selecciona todos los elementos. Opcionalmente, puede estar restringido a un espacio de nombre especfico o a todos los espacios de nombres.
    Sintaxis: ns| |
    Ejemplo: * se aplicar a todos los elementos del documento.
  • Selector de atributo
    Selecciona elementos basndose en el valor de un determinado atributo.
    Sintaxis: [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value]
    Ejemplo: [autoplay] seleccionar todos los elementos que tengan el atributo autoplay establecido (a cualquier valor).

Anatoma de las reglas de estilo

Ahora que sabemos como seleccionar los elementos a los que queremos aplicar las reglas podemos escribir qu reglas queremos que se apliquen. Para el ejemplo vamos a usar la etiqueta <style>.

<style>
  body {}

  .divClass {}

  #divId {}
</style>

En el ejemplo de arriba vemos tres selectores. El primero es para el elemento body, el segundo para todos los elementos de la clase divClass y el tercero para el elemento que tenga el id: divId. Dentro de los {} vamos a escribir todas las reglas que queremos que se apliquen a esos elementos.

Box Model

Para poder entender y luego manipular la forma en que los elementos HTML aparecen distribuidos en la pgina, tenemos que aprender cmo son representados estos en el browser.

En un documento html cada elemento es representado como una caja rectangular y en CSS cada una de estas cajas tiene 4 capas que podemos manipular. Esto se conoce como Modelo de Caja o Box Model.

Yendo desde afuera hacia adentro, las capas son:

  • margin: el espacio que separa al elemento de los otros elementos. Si los pensamos como cajas, es el espacio entre las cajas.

  • border: el borde de la caja. Podemos hacerlo visible con diferentes grosores, estilos y colores, como ya hicimos varias veces en ejercicios anteriores.

  • padding: el espacio entre el borde de la caja y su contenido. En la metfora de la caja, podramos por ejemplo tener una caja grande con algo chiquito adentro, osea que habra mucho padding.

  • content: el contenido de la caja. Por ejemplo el texto en un h1, otros tags anidados, etc, todo lo que est contenido en el elemento.

height y width

Podemos decirle al navegador exactamente qu tan alto y ancho queremos que sea nuestro elemento (contenido), esto se usa en div, img y otros elementos basados en la altura (para determinar el tamao del texto, necesitaremos usar un propiedad de estilo diferente). Los valores de tamao pueden estar en muchas medidas diferentes, pero el ms comn es el pxel px.

div {
  height: 400px;
  width: 400px;
}

margin

El margen es el rea transparente alrededor del elemento que deseas que no choque con nada. Es la capa ms externa en el Modelo de caja.

border

Borde establecer un borde alrededor de su elemento, puedes determinar el tamao, color y estilo del borde. Puede encontrar una lista de estilos de borde aqu: https://developer.mozilla.org/en-US/docs/Web/CSS/border. El borde est fuera del padding, pero dentro del margen.

div {
  border: 1px solid black;
}

padding

El padding es el area transparente entre el borde y el contenido, es similar al margen, pero para adentro

Clculo del box model

Cuando establecemos el alto y el ancho de un elemento a traves de la regla css height y width, slo estamos configurando el contenido. Para calcular la altura y el ancho reales, tenemos que tener en cuenta el padding, el borde y el margen.

  • padding es un rea transparente alrededor del contenido.
  • border se envolver alrededor del relleno
  • margin es el rea transparente ms externa que envuelve toda la caja.

Por ejemplo. Si establecemos la altura del contenido en 20 px y el ancho en 20 px, el relleno en 5 px, el borde en 1 px y el margen en 10 px.

Altura real = 25px (contenido) + 2 * 5px (relleno, cada lado) + 2 * 1 (borde de cada lado) + 2 * 10 (margen, cada lado) = 57px

Ancho real = 25px (contenido) + 2 * 5px (relleno, cada lado) + 2 * 1 (borde de cada lado) + 2 * 10 (margen, cada lado) = 57px

Saber esto nos ayudar a dimensionar y posicionar nuestros elementos correctamente.

Hojas de estilo externas y el elemento <link>

Hemos explicado cmo usar el elemento html <style>. Esto est bien si tiene una pgina web muy pequea y un estilo mnimo, pero la mayora de las pginas comenzaran a sentirse abarrotadas muy rpidamente si incluimos todo nuestro CSS en el HTML. Afortunadamente, tenemos una solucin para eso, hojas de estilo externas y el elemento <link>.

Una hoja de estilo externa es simplemente otro archivo con el tipo de archivo .css. Convencionalmente, este archivo se llama algo as como style.css. Podemos tomar todas las reglas de estilo que escribimos entre las etiquetas <style> y transferirlas directamente al archivo css. No necesitamos incluir nada ms, solo las reglas de estilo.

Una vez que tengamos una hoja de estilo externa creada, necesitaremos asegurarnos de que el navegador lea ese archivo y aplique las reglas a nuestra pgina. Le decimos al navegador que busque ese archivo utilizando el elemento <link>. Podemos eliminar las etiquetas <style> y en su lugar agregar el elemento <link>. Dentro del elemento de enlace, necesitaremos proporcionar la ubicacin y el tipo de archivo que estamos vinculando. Utilizaremos dos banderas, la bandera rel y la bandera href.

La bandera rel solo le dir al navegador qu tipo de archivo es y cmo procesarlo. En nuestro caso lo configuraremos como hoja de estilo

La bandera href le dir al navegador dnde encontrar el archivo. Si el archivo est en la misma carpeta que nuestro archivo html, podemos configurarlo en: ./styles.css (esta ruta ser relativa)

<link rel = "stylesheet" href = "./ styles.css" />

Ahora que tenemos nuestra hoja de estilo externa vinculada a nuestro archivo HTML, deberamos ver las reglas de estilo que establecemos reflejadas en nuestra pgina.
